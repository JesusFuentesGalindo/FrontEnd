Proceso de Extracción de Datos Para Generación de Reportes

Generales

Objetivo: Descargar los datos necesarios desde la base de datos para la creación de los reportes de I&C.

Justificación: Este proceso es necesario debido a que se necesita extraer de las diferentes bases de datos la información necesaria para la realización de los reportes del área de Información y Control.

Resultado: Archivo de texto o Excel con los datos que se extrajeron desde la base de datos.

Entrada:

  • Conexión a base de datos SQLServer
  • Consultas SQL

Particulares

Conexión a base de datos

Es necesario contar con las conexiones a los siguientes servidores:

  • SKYMXSICO1
  • SKYMXMULTI1

Consultas

Las consultas necesarias se encuentran en: "\\Dt-val-98\I&C\Querys\Mis Query's -2017 y 2018"

Proceso

No Ingresados (No Retornados) - 01

Directorio de Extracción: \\dt-val-98\I&C\Reportes Operativos\Ventas\No Retornados Mensual\Extracciones

Extracción Master
  1. Conectarse al servidor SKYMXSICO1
  2. Abrir Consulta: No Retornados Master 2018.sql
  3. Seleccionar consulta de Master
  4. Seleccionar base de datos master
  5. Seleccionar opción guardar consulta en archivo
  6. Ejecutar consulta
  7. Guardar archivo en Directorio de Extracción con el nombre 'No retornados Master_[Fecha Extracción].rpt'
  8. Cerrar consulta
  9. Desconectarse del servidor SKYMXSICO1
  10. Fin
Extracción Fuerza Propia
  1. Conectarse al servidor SKYMXMULTI1
  2. Abrir Consulta: No Retornados Master 2018.sql
  3. Seleccionar consulta de Distribuidor
  4. Seleccionar base de datos St_Multiregional
  5. Seleccionar opción guardar consulta en archivo
  6. Ejecutar consulta
  7. Guardar archivo en Directorio de Extracción con el nombre 'No retornados Dist_[Fecha Extracción].rpt'
  8. Cerrar consulta
  9. Desconectarse del servidor SKYMXMULTI1
  10. Fin

      2. Obtener ID activación de la extracción de calidad Master
      3. En la consulta Master remplazar el id_activacion por los obtenidos del archivo de extracción de calidad Master
      4. Selecciona la base de datos relacionada al Master
      5. Selecciona la consulta Master
      6. Ejecutar la consulta (F5)
    2. Copiar los datos de la consulta al libro creado de Excel
    3. Hacer los pasos anteriores por cada Master de la tabla Distribuidores
  5. Guardar archivo en Directorio de Extracción con el nombre 'Motivos de rechazo Calidad_[Fecha extracción].xlsx'

Incumplimiento a la Normatividad (Alteraciones) - 11

Directorio de Extracción: \\dt-val-98\I&C\Proceso de Penalizaciones\Alteraciones_Falsificación\Archivos de trabajo\Extracciones

Extracción Master y Fuerza Propia
  1. Conectarse al servidor SKYMXSICO1
  2. Abrir Consulta: Alteraciones.SQL
  3. Crear un libro nuevo de Excel
  4. Consolidar en Excel
    1. Ejecutar consulta
      1. Cambia el ID del Distribuidor (id_master en la consulta)
      2. Selecciona la base de datos relacionada al Master
      3. Seleccionar consulta Master
      4. Ejecutar la consulta (F5)
    2. Copiar los datos de la consulta al libro creado de Excel
    3. Hacer los pasos anteriores por cada Master de la tabla Distribuidores
  5. Cerrar consulta
  6. Desconectarse del servidor SKYMXSICO1
  7. Conectarse al servidor SKYMXMULTI1
  8. Abrir Consulta: Alteraciones.SQL
  9. Consolidar en Excel
    1. Ejecutar consulta
      1. Seleccionar consulta del Distribuidor
      2. Selecciona la base de datos St_Multiregional
      3. Ejecutar la consulta (F5)
    2. Copiar los datos de la consulta al libro creado de Excel
  10. Guardar archivo en Directorio de Extracción con el nombre 'Extracción de Alteraciones al [Fecha Extracción].xlsx'
  11. Fin
