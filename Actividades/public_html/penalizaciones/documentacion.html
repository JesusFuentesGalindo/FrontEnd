Base de Datos Penalizaciones

Generales

Objetivo: Desarrollar una base de datos que sirva como respaldo de los datos resultado del proceso de penalización, del área de información y control.

Justificación: Mantener los datos resultado del proceso de penalización bajo un formato que garantice la seguridad, consistencia y persistencia de los mismos.

Resultado: Base de datos bajo la plataforma Access.
